# Expenses Tracker System

This is an Expenses Tracker desktop application that allows users to record and monitor expenses on a **daily, weekly, monthly, and yearly** basis.

The main purpose of this application is to help **small businesses and individuals** track their expenses efficiently and support **better financial decision-making**.

## Features
- Record daily expenses
- View expenses by week, month, and year
- Automatic total calculation using SQL queries
- Simple and user-friendly desktop interface

## Technologies Used
- **Python** – core programming language
- **Tkinter** – GUI (desktop application interface)
- **SQLite3** – local database for storing expense records

## Learning Objectives
This project was built to:
- Practice **data structure operations**
- Improve understanding of **SQL queries**, including:
  - Filtering data
  - Using aggregate functions like `SUM`
- Gain hands-on experience in building a **real-world desktop application**

Desktop Application
